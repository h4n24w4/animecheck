# アニメ検索くん
'アニメ検索くん' is using Vue.js, Vuetify

## Project setup
```
git clone https://github.com/h4n24w4/animecheck.git
cd animecheck
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Package
```
vue: 2.6.10
vuetify: 1.5.14
axios: 0.18.0
```
